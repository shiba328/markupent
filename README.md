# markupent-webpack
 webpack development to generate static markup pages using pug and sass ...and more.

# Installation
```
$ git clone https://github.com/shiba328/markupent.git
```

# Usage
```
$ npm i
```

***Commands:***
```
init    
```
***Options:***
```
-v, --version    output the version number
-h, --help       output usage information
```
